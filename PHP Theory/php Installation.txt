Php Installation
================
To install PHP, we have to install AMP (Apache, MySQL, PHP) software stack. 
It is available for all operating systems. 
There are many AMP options available in the market that are given below:

1.WAMP for Windows
2.LAMP for Linux
3.MAMP for Mac
4.SAMP for Solaris
5.FAMP for FreeBSD
6.XAMPP (Cross, Apache, MySQL, PHP, Perl) for Cross Platform: 
  It includes some other components too such as FileZilla, OpenSSL, Webalizer, Mercury Mail, etc.
